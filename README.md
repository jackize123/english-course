# 我的英文自學路線圖 📖

從自然發音一路讀到學測程度的完整自學課程,單一 HTML 檔、離線可用。

## 內容

- **六階段學習路線圖** — 從第 1 階發音築基到第 6 階學測進階,含里程碑檢核
- **發音教室** — 26 字母、五短母音、CVC 拼讀、雙字母組合、聽力測驗(內建語音朗讀)
- **教材庫** — 21 課文法(觀念/規則/例句朗讀/小提醒)、105 題練習、主題單字、分級閱讀
- **273 天每日課表** — 39 週,每天都有具體任務,可打勾追蹤
- **進度備份/還原** — 換裝置時可匯出匯入

進度儲存在瀏覽器 localStorage,不會上傳到任何地方。
